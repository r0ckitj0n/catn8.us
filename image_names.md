## Image Guidelines
- All images should be in JPG format
- Dimensions: 400x200 pixels
- File size: Optimize for web (under 100KB each)
- Style: Consistent with the website's playful and educational theme
- Colors: Use the website's color palette (primary: #FF6B6B, secondary: #4ECDC4, accent: #FFE66D)
- Accessibility: Ensure good contrast and readability
- Branding: Include relevant logos where appropriate
